    SCROLLABLE

  1. the html iframescrolling Attribute is used to specify whether the scrollbar will be displayed or not in the Iframe Element.
  2. On an iFrame of type Expand to Fit content, when we set the attribute "scrolling=yes", it enables both horizontal and vertical scroll bars.

    3. ALLOW FULL SCREEN

  3. Published: If you are having trouble with full-screen mode not working in iFrame embedding, you can resolve this issue by adding the allowfullscreen attribute to the iFrame tag
  4. You can simply set the iframe height and width in vh (viewport height) and vw (viewport width) units respectively to make it cover full-screen with a height and width of 100%.

    5. META TAGS VIEWPORT

  5. A viewport meta tag is HTML (HyperText Markup Language) code that tells browsers how to control viewport dimensions and scaling.
  6. It's a key ingredient of responsive web design and ensures your content is easy to view. A viewport is the user's visible area of a webpage.
